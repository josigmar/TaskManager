# 🚀 TaskManager - AL Project for Business Central

This is a practice exercise developed in AL for Microsoft Dynamics 365 Business Central. It focuses on task management within the ERP environment.

## 🛠️ Features
* **Task Management:** Create, edit, and delete custom tasks.
* **Integration:** Seamless connection with standard BC tables.
* **UI/UX:** Optimized List and Card pages for a better user experience.

## 📦 Prerequisites
* **Visual Studio Code** with the **AL Language** extension.
* Access to a **Business Central Sandbox** (Cloud or Docker).

## 🚀 Installation & Configuration
1. Clone the repository:
   `git clone https://github.com/josigmar/TaskManager.git`
2. Open the folder in VS Code.
3. Download symbols (`Ctrl + Shift + P` -> `AL: Download symbols`).
4. Publish to your environment (`F5`).

## 💻 Tech Stack
* **Language:** AL
* **Platform:** Microsoft Dynamics 365 Business Central
* **Runtime:** 12.0 (Adjust according to your app.json)

# 🚀 TaskManager - Proyecto AL para Business Central

Este es un ejercicio de práctica desarrollado en AL para Microsoft Dynamics 365 Business Central, enfocado en la gestión de tareas dentro del entorno del ERP.

## 🛠️ Funcionalidades
* **Gestión de Tareas:** Creación, edición y borrado de tareas personalizadas.
* **Integración:** Conexión fluida con tablas estándar de BC.
* **Interfaz:** Páginas de tipo List y Card optimizadas para el usuario.

## 📦 Requisitos previos
* **Visual Studio Code** con la extensión **AL Language**.
* Acceso a un **Sandbox de Business Central** (Cloud o Docker).

## 🚀 Instalación y Configuración
1. Clonar el repositorio:
   `git clone https://github.com/josigmar/TaskManager.git`
2. Abrir la carpeta en VS Code.
3. Descargar símbolos (`Ctrl + Shift + P` -> `AL: Download symbols`).
4. Publicar en el entorno de desarrollo (`F5`).

## 💻 Tecnologías utilizadas
* **Lenguaje:** AL
* **Plataforma:** Microsoft Dynamics 365 Business Central
* **Runtime:** 12.0 (Ajustar según tu app.json)
